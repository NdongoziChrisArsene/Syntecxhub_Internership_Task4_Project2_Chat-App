⚠️  IMPORTANT — READ THIS BEFORE BUILDING ⚠️

This folder is missing ONE file that only YOU can provide:

    google-services.json

This file is unique to your Firebase project (smart-chat-app-39fe1).
It cannot be generated or faked — it must come from your Firebase console.

HOW TO GET IT:
1. Go to https://console.firebase.google.com
2. Open your project (smart-chat-app-39fe1)
3. Click the gear icon ⚙️ → Project settings
4. Scroll down to "Your apps" → find your Android app
5. Click "google-services.json" download button
6. Place the downloaded file in this same folder (app/)
   so it sits next to build.gradle.kts

Once you've added that file, you're ready to open the project
in Android Studio and sync Gradle.
